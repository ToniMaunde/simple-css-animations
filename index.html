{ Quick introduction }

I decided to create this open source project to share the CSS animations that I’ve learned and catalogue, in a way, my learning experience hoping that it might benefit someone else.

{ The gist of it }

Down below you’ll find the available CSS animations alongside the markup (HTML) and styling (CSS). These can be easily copied and altered to suit your needs. More details on the structure of the code can be found on the repository of this project on GitHub .

{ The CSS Animations }

Finally, here they are. I hope you can find it useful.
